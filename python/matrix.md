# Матрицы
---
Матрица - прямоугольная таблица, составленная из элементов одного типа (int, str etc.) Фактически - список из вложенных списков.

### Создание матрицы

    cols = 3      # rows - количество строк, cols - количество столбцов
    rows = 2
    matrix = []
    for i in range(cols):
        matrix.append([0] * rows)
Результат:

    print(matrix)
    
    >>[[0, 0], [0, 0], [0, 0]]
